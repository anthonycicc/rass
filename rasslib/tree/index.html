Module rasslib::tree [−] [src]

Contains structs and traits for a Tree structure. Subtrees are of the same type Tree<T>. A visitor-pattern trait is provided which also internally used for the TreePrinter.

Path represents a path through the tree to a specific not. Finding an element returns a vector of paths.
